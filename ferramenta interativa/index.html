Flashcards Neon de Estudos

O que é HTML?
HTML é uma linguagem de marcação para criar páginas web.
O que é CSS?
CSS é uma linguagem de estilos para definir a aparência das páginas web.
O que é JavaScript?
JavaScript é uma linguagem de programação que permite interatividade nas páginas web.
O que é uma variável?
Uma variável armazena dados para serem usados e manipulados pelo código.
O que é uma função?
Funções são blocos de código que executam uma tarefa específica.
O que é uma condição?
Uma condição é uma estrutura que permite executar blocos de código diferentes com base em testes lógicos.
